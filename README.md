# MQTT CO2 K30

Simple project to get [K30 CO2 sensors](https://www.co2meter.com/products/k-30-co2-sensor-module) online using MQTT.

Designed to work with [mqtt.chem.wisc.edu](https://mqtt.chem.wisc.edu/).
